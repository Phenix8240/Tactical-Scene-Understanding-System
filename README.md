# 🧠 AI-Powered Tactical Scene Understanding System

An intelligent computer vision system that analyzes military-style images to extract tactical insights such as soldier count, weapon detection, formation patterns, environment type, and time of day.

---

## 📌 Overview

This project uses deep learning and computer vision techniques to transform raw images into actionable intelligence.

The system processes an input image and automatically identifies:

- 👤 Number of soldiers
- 🔫 Weapons detected (e.g., rifles)
- 🧍 Formation type (tight / loose / spread)
- 🌍 Environment (urban / forest / desert)
- 🌗 Time of day (day / night)
